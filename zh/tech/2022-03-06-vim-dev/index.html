极简的 Vim 配置

  1. fzf.vim 使用 FZF 作为搜索前端，调用 riggrep 和 fd 来检索。主要优势是模糊查找（关键词不需要按顺序和连续输入）和查询速度。
  2. fern 是一个树状浏览 Plugin，和 IDE 常见的 Project Explorer 或者 File Drawer 类似。主要优势是现代的功能，比如异步渲染，支持插件扩展等。
  3. vim-lsp 是一款 LSP 插件。使用 vim-lsp-settings 可以一键安装大多数语言。
  4. asyncomplete.vim 与 vim-lsp 同属一个作者，推荐搭配使用。和其他主流补全（COC 或者 YouCompleteMe）相比，属于轻量但是n高性能，0延迟的补全插件。
